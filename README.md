# Website Traffic Sources Analysis
## Project Overview

This project analyzes website traffic data using Python and visualizes important traffic insights through graphs and dashboard summaries.  
It also performs simple future traffic prediction using traffic analysis techniques.

The project helps understand:
- Visitor trends
- Traffic source performance
- Bounce rate analysis
- User engagement
- Future website traffic prediction

The project is built using:
- Pandas
- Matplotlib
- Scikit-learn
  
# Features
- Data Cleaning and Preprocessing
- Monthly Website Traffic Analysis
- Traffic Source Visualization
- Bounce Rate Analysis
- Session Duration Analysis
- Device Type Analysis
